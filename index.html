Google Location History Browser

Visualize and explore your Google Location History data. Note: Google Location History service has been deprecated, but you can still export your historical location data using Google Takeout.

Getting Started

  1. Export your location data from Google Takeout (select "Location History" as JSON format)
  2. Extract the downloaded archive (ZIP file)
  3. Navigate to the "Location History" folder inside the extracted archive
  4. Select the Records.json file (or multiple files if your history is split)
  5. Load the file(s) using the "Load History" button
  6. Use the date range slider to filter locations by time

Tips

  • Use Tab key to select slider handles, then Left/Right arrows to move by hours
  • Hold arrow keys for accelerated movement
  • Multiple map styles available for different visualization needs

Privacy: All processing happens locally in your browser. No location data is sent to any server.
