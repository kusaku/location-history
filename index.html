Location History Visualizer

Visualize your Google Location History data with interactive heatmaps. Supports both original JSON files and compressed .json.gz files for faster loading.

Quick Start

  1. Export location data from Google Takeout (JSON format)
  2. Extract ZIP and find Records.json in "Location History" folder
  3. Click "Load History" and select your file(s)
  4. Use the date slider to filter by time, customize map styles and gradients

Tips

  • Tab + arrow keys to navigate sliders (hold arrows for acceleration)
  • Try different map styles and gradients to find your preferred visualization
  • Adjust radius and blur sliders to fine-tune heatmap intensity

Privacy: All processing happens locally. No data is sent to any server.
